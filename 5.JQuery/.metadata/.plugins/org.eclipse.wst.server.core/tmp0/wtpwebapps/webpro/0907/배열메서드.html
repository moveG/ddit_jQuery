배열 메서드

indexOf(item) : 배열에서 item의 위치(배열의 인덱스)를 리턴, 존재하지 않으면 -1을 리턴

indexOf(item, start) : 배열에서 item의 위치를 리턴(start에서 검색 시작)

lastIndexOf(item) : 배열에서 item의 위치를 리턴(뒤에서부터 검색)

lastIndexOf(item, start) : 배열에서 item의 위치를 리턴(뒤에서부터 검색, start에서 검색 시작)


              






배열 메서드

push(a, b, c, ...) : 배열의 끝에 a, b, c, ...를 추가

pop() : 배열의 끝의 요소를 제거


    




배열 메서드

shift() : 배열의 첫 요소를 제거

unshift(a, b, c, ...) : 배열의 처음에 a, b, c, ...를 추가


    




배열 메서드

slice(start, end) : start ~ end 사이의 요소를 추출해서 새로운 배열을 생성(end는 포함되지 않음)

splice(index, n, a, b, c, ...) : index 위치에서 n개를 삭제하고, a, b, c, ...를 추가한다.

(a, b, c,...이 없으면 삭제만 수행함, n이 0이면 추가만 수행함)


    




배열 메서드

a.concat(b, c) : a배열과 b배열과 c배열을 하나로 합친다.

join() : 하나의 배열을 문자열로 변환하면서 사이에 ','를 추가한다.

join("/") : 하나의 배열을 문자열로 변환하면서 사이에 '/'를 추가한다.


    




배열 메서드

reverse() 배열의 순서를 뒤집는다.

sort() : 배열을 사전순으로 정렬한다.

사전순O(1, 11, 2, 3), 오름차순X(1, 2, 3, 11)

sort(function(a, b) {return a - b}) : 배열을 오름차순으로 정렬한다.
