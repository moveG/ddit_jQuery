String객체

str = "우리 나라 좋은 나라 대한 민국 만세";

charAt(index) : index위치의 문자를 리턴

charCodeAt(index) : index위치의 문자의 유니코드를 리턴


    




String객체

str = "우리 나라 좋은 나라 대한 민국 만세";

indexOf(item, start) : start위치에서부터 item의 위치를 찾아서 인덱스 주소를 리턴(start는 생략이 가능함)

lastIndexOf(item, start) : start위치에서부터 item의 위치를 역방향으로 찾아서 인덱스 주소를 리턴(start는 생략이 가능함, 뒤에서부터 X번쨰가 아님)


    




String객체

replace(search, value) : search의 문자열을 value의 문자열로 대체

match(regexp) : 정규식으로 검색하여 일치하는 결과를 배열로 리턴, 없으면 null을 리턴



    




String객체

substing(from, to) : 두 위치 사이의 문자열을 추출, end는 제외

slice(start, end) : : start와 end 사이의 문자열을 추출, end는 제외

substr(start, length) : start에서 length개수만큼 추출


         





String객체

split(seperator, limit) : 구분자를 기준으로 문자열을 분리하여 limit수 만큼만 배열로 리턴함(limit를 생략하면 전체 문자열이 대상)
