Window객체

open("실행할 window", "name/target", "specs")

- target : _blank, _self

- name : 논리적으로 표현되는 이름, 물리적 표현이 아니므로 표시되지 않는다.

- specs : 여러가지의 속성을 설정할 수 있음(width, height, top, left 등)
