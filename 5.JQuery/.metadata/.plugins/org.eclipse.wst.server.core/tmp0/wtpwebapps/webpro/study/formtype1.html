입력요소 : Input타입의 종류
			- text : 기본입력 요소, 문자, 숫자, 날짜 등의 모든 성질의 데이터 입력이 가능함
			- password : 비밀번호 입력 
			- date : 날짜형식의 데이터 
			- submit : 전송, 제출
			- button : 초기기능 없음, 원하는 기능을 부여하기 위해서 JavaScript가 필요함
		








			<input type = "button" value = "제출1" onclick = "proc()">
			- 우리가 만든 proc를 실행시켜라
			<input type = "button" value = "제출2" onclick = "alter('Hello~~~')">
			- onclick하면 altet(창을 띄워라)해라. 그 안의 메시지는 Hello~~~이다(메시지에는 작은 따옴표를 넣어줌)
