id 접근 : document.getElementById('x')

class 접근 : document.getElementsByClassName('abc') - 배열로 리턴함

태그이름 접근 : document.getElementsByTagName('div') - 배열로 리턴함





id 접근 : document.querySelector('#x')

class 접근 : document.querySelector('.abc') - 첫번째만 리턴함

태그이름 접근 : document.querySelector('div') - 첫번째만 리턴함



모든 class 접근 : document.querySelectorAll('.abc') - 배열로 리턴함

모든 태그이름 접근 : document.querySelectorAll('div') - 배열로 리턴함
